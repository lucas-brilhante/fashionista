## Fashionista E-commerce

Fashionista é um projeto feito com ReactJS, que tem como objetivo a construção do front-end de um E-commerce de acessórios femininos.

Os componentes utilizados no projeto foram:
*Create React App
*React Hooks
*React Router Dom
*Redux
*Redux Thunk
*React Icons
*Styled Components
